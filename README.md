# Emotion Detector Application

## 📌 Project Overview
This is an AI-based web application that detects emotions from text input using IBM Watson NLP (Natural Language Processing) library. The application analyzes the provided text and returns scores for five emotions: **anger, disgust, fear, joy, and sadness**, along with the dominant emotion.

## 🚀 Features
- Detects emotions from user-provided text
- Returns emotion scores for anger, disgust, fear, joy, and sadness
- Identifies the dominant emotion
- Web-based user interface using Flask
- Error handling for invalid/blank inputs
- Unit testing for application functions
- Static code analysis with PyLint

## 🛠️ Technologies Used
- **Python 3.x**
- **Flask** - Web framework
- **IBM Watson NLP** - Emotion detection API
- **Requests** - HTTP requests library
- **UnitTest** - Unit testing framework
- **PyLint** - Static code analysis
